React
  • Front end Javascript library to build user interfaces
  • Makes it easier to create responsive web pages
  • Developed by Facebook
Fundamentals
Component
  • Fundamental building block of React
  • Lets you split up the UI into independent, reusable pieces, and think about each piece in isolation
  • Conceptually they are like JavaScript functions
    • Accept arbitrary inputs called "props"
    • Return React elements that describe what should appear on the screen
Component Example
Props
  • Passed down to component from parent component
  • Should be immutable or "pure"
    • Read-Only, never modified
  • All React components must act like pure functions with respect to their props
    • function sum(a,b){
       return a + b;
    }
    function withdraw(account, amount){
       account.total -= amount;
    }
State
  • Represents the internal state of the component
  • When a components state data changes, the component is re-rendered
Virtual DOM
  • Document Object Model (DOM)
  • The real DOM is the structure of a webpage
Virtual DOM
  • DOM manipulation os the heart of the modern, interactive web
  • Unfortunately, it is a lot slower than most JavaScript operations
  • Slowness is made worse by the fact that most JavaScript frameworks update the DOM much more then they have to
  • Example: updating a list
Virtual DOM
  • To address this inefficiency, React popularized the virtual DOM for efficient re-rendering of the actual DOM
  • The virtual DOM is a lightweight copy of the real DOM
  • React aims to re-render the virtual tree only when the state changes
  • Once the virtual DOM has updated, React compares the virtual DOM with a virtual DOM snapshot before the state change
  • It then compares which objects have been changed and only updates those objects on the real DOM
JSX
  • JavaScript extension
  • Allows us to combine Javascript and HTML
PROS
  • Fast and efficient through the use of a virtual dom
  • Makes writing Javascript easier with JSX
  • Easy to read and understand code
  • React dev tools (Chrome extensions makes debugging easier)
CONS
  • React is only the view layer
  • Learning curve
